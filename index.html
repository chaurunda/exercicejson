Bienvenue dans ce magnifique exercice pour vous apprendre à manipuler du JSON.

Le but recherché est de pouvoir récupérer du JSON avec de AJAX et de manipuler le DOM pour en faire ce que vous souhaitez

Qu'est-ce que le JSON

Selon le Mozilla Developper Network :

JSON (JavaScript Object Notation) est un format d'échange de données. Il ressemble fortement à un sous-ensemble de la syntaxe JavaScript, bien que ce ne soit pas exactement un sous-ensemble strict. Ce langage est utile pour les applications écrites en JavaScript comme les sites web ou les extensions. Par exemple, on peut stocker des informations dans un cookie en JSON ou bien enregistrer des préférences avec un objet JSON.

JSON permet de représenter des nombres, des booléens, des chaînes de caractères, la valeur null, des tableaux (qui sont des séquences ordonnées de valeurs) et des objets (un ensemble de correspondances chaîne-valeur) composés de ces valeurs (ou d'autres tableaux ou objets). Il ne permet pas, nativement, de représenter des données plus complexes comme les fonctions, les expressions rationnelles, les dates ou autre. (Les objets Date sont traduits vers une chaîne de caractères selon un format ISO, cela permet pour certains cas, d'assurer le transport de ces données.) Si vous avez besoin de préserver ces valeurs et de représenter des valeurs d'autres types de données, vous pouvez les transformer lors de la conversion en chaîne de caractères et/ou les re-transformer avant de les convertir en objets JavaScript.

Pour faire simple, le JSON (JavaScript Object Notation) est un format reprenant l'objet javascript.

En effet la notation du json ce constitue de la façon suivante :

            {
                key : value,
                anotherKey : anotherValue
            }

L'objectif va être donc de savoir appeller et manipuler le DOM pour y injecter nos différentes données.
Avant de commencer, je vais vous donner quelques exemples afin de comprendre la manipulation à effectuer.

Exemples

Avant-propos

Dans cette exemple je vais partir du principe que vous avez déjà une page html remplis (ou non) ainsi que jquery d'intégré à votre page.

Partons du principe que nous avons les fichiers suivants

index.json

            {
                "text": "Salut les loulous"
            }

index.html

            
            <html lang="fr">
            <head>
                <meta charset="UTF-8">
                <title>Document>/title>
            </head>
            <body>
                <p>Bonjour à tous je voulais vous dire ceci : </p>
                <div id="message"></div>
                <script src="https://code.jquery.com/jquery-1.11.3.min.js"></script>
            </body>
            </html>

Comme vous pouvez le voir il c'est simple à comprendre.

Maintenant, regardez comment, grace à jQuery je vais récupérer mon fichier JSON et l'intégrer dans ma page

script.js

            $(document).ready(function(){
                $.getJSON('index.json', function(data){
                    console.log(data);
                    $('#my-message').text(data.text);
                });
            });

Comme vous pouvez le voir, data récupère tout notre contenu de notre fichier index.json. Pour récupérer la valeur de la clé text, j'ai écris data.text.
